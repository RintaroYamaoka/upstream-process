# 責任分担表(RACI) — <プロジェクト名>

> 記入要領: **作業の分担ではなく判断の分担**を書く。
> 決定者の判定基準: 「その判断を間違えたとき、最初に気づけるのは誰か」。
> 安定した割当だけをこの表に書く。案件ごとの割当は作業依頼書の「あなたが決めること」欄が主で、この表は従。
> D=決定 C=相談される I=報告を受ける

## 運用ルール

- 決定を含む変更(PR)は、**その決定の持ち主が merge する**(merge は署名 — 押した人がその決定を承認したことになる)
- 他人の判断領域に踏み込む必要が生じたら(緊急対応等)、**仮置きラベルを貼る**: 「これは◯◯の領域だが△△の理由で仮置きした。確定は◯◯さんが行う」。貼らないラベルは黙って決定になる
- 決定者が負担過多のとき、引き取るのは**作業**(実装)であって**判断**ではない。「この案でいくか?」を聞く時間は 5 分で済む
